Detailed Description

Exported dialog box class for entering a machine seed

You can't open a Dialog Box from a remote DLL, so this class is a wrapper that creates and calls DoModal on the dialog box in this class.

You are not allowed to change this class in any way!

AFX_EXT_CLASS is a Microsoft directive that indicates that this class can be used outside the DLL. Do not change it or add it to any other classes.
